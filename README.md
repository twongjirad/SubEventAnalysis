# SubEventAnalysis


## Dependencies

* pylard
* virtualenv

## Installation

* setup virtualenv
```
virtualenv --system-site-packages env
```
* install pylard. in pylard directory
```
python setup.py install
```
* build and install pysubevent. in this directory
```
python setup.py install
```

## setup environment

* use setup scrpit
```
source setup.sh
```
